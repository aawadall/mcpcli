Getting Started

Get up and running in minutes

1

Install

Download the latest release for your platform

# Linux/macOS curl -L https://github.com/aawadall/mcpcli/releases/latest/download/mcpcli-linux-amd64.tar.gz | tar -xz sudo mv mcpcli-linux-amd64 /usr/local/bin/mcpcli
2

Generate

Create your MCP server project

mcpcli generate # Follow the interactive prompts
3

Develop

Start building your MCP server

cd my-mcp-server go run cmd/server/main.go
4

Test

Validate your server works correctly

mcpcli test # Run built-in validation tests
